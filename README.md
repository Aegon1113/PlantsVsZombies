# 植物大战僵尸程序说明
- - -

## 项目背景
很多同学都玩过植物大战僵尸，我们希望通过实现一个简单版本的植物大战僵尸来提高我们App开发的能力。
- - -

## 项目介绍
我们的植物大战僵尸主要分成3个模块：战斗模块、图鉴模块、商店模块。
- - -

### 战斗模块
在这个模块中我们会实现僵尸的产生、行进、攻击，植物的种植、反击、死亡。
- - -

### 图鉴模块
在这个模块，我们会实现植物与僵尸的图鉴显示，介绍相关信息。
- - -

### 商店模块
在这个模块我们会实现一些道具的出售以及氪金的通道。
